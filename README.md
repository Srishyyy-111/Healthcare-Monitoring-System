## Overview
A Python-based healthcare monitoring system that analyzes
basic health parameters and generates alerts if values
are outside the normal range.

## Features
- Monitors blood pressure, heart rate, blood sugar, BMI
- Tracks oxygen level, sleep hours, and water intake
- Handles invalid inputs using error handling
- Generates health alerts and suggestions
